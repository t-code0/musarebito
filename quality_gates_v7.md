# アプリ別合格基準（品質ゲート） v7

## 共通原則
「ターゲットユーザーが課金してまで使いたいと思うか」が唯一の合格条件。

---

## App1：Amazon本物検索

ターゲット：買い物で失敗したくない人・本物志向の購買層

課金する瞬間：
- 「このスコアの根拠が見たい（論文・成分データ）」
- 「新商品が出たらアラートで教えてほしい」
- 「複数商品を並べて比較したい」

合格条件：
- [ ] 本物スコアに必ず根拠（論文名・データ出所）が表示されている
- [ ] 「なぜAランクなのか」を3行以内で説明している
- [ ] Amazonの星評価との違いが明確に伝わる
- [ ] ユーザーが「これ友達に教えたい」と思えるスコア画像が生成できる
- [ ] アフィリリンクがスコア表示の直下にある

不合格条件（即差し戻し）：
- 根拠なしのスコア表示
- Amazonのレビューをそのまま転記しただけ
- 他サービスと見分けがつかないUI

---

## App2：添加物検索図鑑

ターゲット：子育て世代・健康意識の高い30〜50代

課金する瞬間：
- 「この添加物が本当に危険かどうか論文で確認したい」
- 「毎日の食生活スコアを記録・管理したい」
- 「家族全員分の履歴を保存したい」

合格条件：
- [ ] バーコードスキャンから結果表示まで3秒以内
- [ ] 危険度ランク（S〜D）が画面を開いた瞬間に見える
- [ ] 「なぜDランクなのか」添加物名と理由が表示される
- [ ] シェア画像に「危険度〇点」「Dランク」が大きく表示される
- [ ] スキャン履歴が蓄積されて「今月の健康スコア」が見える
- [ ] 「この添加物の論文を見る」ボタンがある（プレミアム誘導）

不合格条件（即差し戻し）：
- スキャン結果が出るまで3秒超
- ランクだけで根拠がない
- 履歴が蓄積されない（毎回使う理由がない）
- シェアしたくなるデザインになっていない

---

## App3：栄養成分検索図鑑

ターゲット：サプリ・プロテインを科学的に選びたい人・栄養士・健康意識の高いママ

課金する瞬間：
- 「この成分の論文を全部読みたい」
- 「複数サプリを比較して最適な組み合わせを知りたい」
- 「成分が変わったらアラートで教えてほしい」

合格条件：
- [ ] 成分名に必ずPubMed論文が紐付いている（最低1件）
- [ ] 「論文8件で効果確認済み」のような信頼指標が表示される
- [ ] 他サイト・Googleで5分調べても出てこない情報密度がある
- [ ] スマホで検索→食品詳細が3タップ以内で到達
- [ ] 検索結果が0.5秒以内に表示される
- [ ] レーダーチャートで栄養バランスが一目でわかる
- [ ] 文科省データ500食品以上が検索可能

不合格条件（即差し戻し）：
- 論文紐付けなしの成分表示
- 「栄養成分が〇mg含まれています」だけの情報
- Googleで「商品名 成分」で検索したら同じ情報が出てくる
- 外国製品ばかりで日本食品が少ない
- UIがぐちゃぐちゃでプロが使えるレベルでない

---

## App4：蒸され人（むされびと）

ターゲット：サウナ好き・ととのいを求める人

課金する瞬間：
- 「このサウナの詳細レポートと混雑時間帯を知りたい」
- 「まだ誰も知らない穴場サウナを先に教えてほしい」
- 「自分のサウナ履歴と健康効果の相関を見たい」

合格条件：
- [ ] AI要約で「この施設の特徴を30秒で把握」できる
- [ ] 本物スコアに「清潔さ・設備・スタッフ・ととのい度」の内訳が見える
- [ ] サウナイキタイにない情報（健康効果論文・混雑傾向）がある
- [ ] 「このサウナ行ってみたい」と思える写真・情報量がある
- [ ] #サウナ #ととのい でシェアしたくなるスコア画像がある

不合格条件（即差し戻し）：
- Googleマップと同じ情報しかない
- AI要約がなく口コミをそのまま羅列している
- サウナ好きが「これは知らなかった」と思う情報がない

---

## App5：近所の本物マップ

ターゲット：外食・美容・医療で失敗したくない人

課金する瞬間：
- 「この地域の全ジャンルの本物ランキングを常に最新で見たい」
- 「口コミの真偽を判定してほしい」
- 「競合店との詳細比較を見たい」

合格条件：
- [ ] AI要約で「この店の特徴を30秒で把握」できる
- [ ] 食べログ・Googleマップと明確に違う「本物スコア」がある
- [ ] 「なぜ本物スコアが高いのか・低いのか」根拠が見える
- [ ] 地域名＋ジャンルで検索してすぐ見つかるSEO構造になっている
- [ ] ユーザーが「この店教えてあげよう」とシェアしたくなる

不合格条件（即差し戻し）：
- Googleマップの評価をそのまま表示しているだけ
- AI要約がなく口コミを羅列しているだけ
- 本物スコアの根拠が見えない
- 食べログで代替できる内容しかない
